--Gerenciador de ranking EAOT 2013--
Arquivos
*index.php - página inicial com a listagem do ranking
*index1.php - formulário para cadastrar novo usuário
*index2.php - autenticação dos dados recebidos
*index3.php - página de sucesso pós autenticação
*index4.php - página de fracasso pós autenticação
*conexao.php - cria a conexão com o banco de dados
*estilo.css - estilo das páginas
*banco.sql - arquivo para gerar as tabelas no MySQL
--SpryAssets
  Arquivos .js para validação dos campos no formulário

Como instalar:
1. Crie o banco de dados com o arquivo banco.sql
2. Configure o arquivo conexao.php
3. Reinicie o apache e acesse o index.php
